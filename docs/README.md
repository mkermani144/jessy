# Jessy Docs

This folder contains practical documentation for extending and operating Jessy.

## Documents

- `docs/PROCEDURE.md`: End-to-end runtime procedure of a scan run.
- `docs/ARCHITECTURE.md`: Hexagonal architecture, boundaries, and important files.
- `docs/ADDING_PLATFORM.md`: Step-by-step guide to add a new job platform.
- `docs/OPERATIONS.md`: Commands, config notes, logging/troubleshooting workflow.

## Suggested reading order

1. `docs/ARCHITECTURE.md`
2. `docs/PROCEDURE.md`
3. `docs/ADDING_PLATFORM.md`
4. `docs/OPERATIONS.md`
